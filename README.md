# Салам Тур — варфрейм лендинга

Интерактивный варфрейм одностраничного сайта туркомпании «Салам Тур» (горный Дагестан, ИП Мухудинов Г.А.).

**Live:** https://markinscompany-commits.github.io/salam-tour/

## Структура

- `index.html` — основной варфрейм со всеми блоками (hero, квиз, каталог, программа по дням, гиды, отзывы, FAQ, индивидуальный тур, галерея)
- `privacy.html` — политика конфиденциальности
- `404.html` — страница для битых ссылок

## Что внутри

13 блоков, переключатель палитр (A/B/C) в правом нижнем углу, модальные окна для всех CTA (заявка, описание тура), маска телефона, валидация форм, cookie-баннер.

Один self-contained HTML-файл, vanilla JS без зависимостей, фавикон inline-SVG.
